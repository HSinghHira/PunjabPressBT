Installation

This Theme is made for Blogger platform only. Theme installation on blogger is very easy task. A beginner with Zero knowledge can easily install theme on Blogger.

Step By Step

Here is a brief step by step on how to install Blogger Theme:

  1. Unzip the “Punjab Press.zip”.
  2. It will contain “Punjab Press.xml” File.
  3. Drop it to Desktop.
  4. On Blogger Dashboard Click Template.
  5. Then click Backup/Restore (Top Right).
  6. Now Choose File
  7. Upload “Punjab Press.xml” file from Desktop.
  8. Then Click Upload.
  9. Done.

Image Tutorial

Default Settings

After installing the theme, this it the minimum setting to make all widget works fine in this themes.

Embedded Comment

This to make comment area like the Demo and make Emotion show

  1. On Blogger Dashboard Click Settings
  2. Click Posts and Comments.
  3. in Comment Location Choose Embedded
  4. Save settings

Blog Feed

This to make recent comment, Search Result, Featured Post, and Post by Label work fine

  1. On Blogger Dashboard Click Settings
  2. Click Other.
  3. in Site Feed Allow Blog Feed Choose Full Or Short
  4. Save settings

Blog Description

This is to make your Blog for better SEO.

  1. On Blogger Dashboard Click Settings
  2. Click Search preferences.
  3. Enable Search Description
  4. Write Suitable Search Description for your Blog
  5. Save changes

NOTE:
1. If yours post have image uploaded on blogspot you can choose SHORT. but if yours post just have image from third party image hosting you must choose Full.
2. Blogger feed must be not redirect to other feed site like Feedburner
3. Yours blog or website not Private
4. Write Post Description also while write new post.

Set Mobile Template

Edit Meta Tags

These are some Meta Tags that are important for Blog's SEO. Read below that how to edit them carefully. (Case Senstive)
  1. On Blogger Dashboard Click Template
  2. Click on Edit HTML
  3. Search for "<!-- MORE TAGS (EDIT TAGS ACCORDING TO YOU)-->"
  4. Replace/Edit Meta Content according to you i.e. FB_ID with 100001475504777 (read below tables for more)
  5. After Editing Click Save Template

Some Important Meta Tags

Property Description
FB_APP_ID Your Facebook App ID (Required if using Facebook Comment Box)
FB_ID Your Profile Numbric ID (Required if using Facebook Comment Box)
GOOGLE+_PROFILE_LINK Your Google Plus Profile Link
SITE_KEYWORDS Site Keywords for your Blog i.e. If Blog is about Blogging then Keywords is Blogging (Seprated by Commas)
YOUR_NAME Replace with Blog's Author Name
TWITTER_USERNMAE Your Twitter Username
USERNAME Your Facebook Page or Profile Username
AUTHOR_NAME Replace with Blog's Author Name

Chat Configurations

Chat Turn On/Off

This is to make Chat turn On or Off for your Blog.

  1. On Blogger Dashboard Click Layout
  2. Click on Chat ON/OFF Widget below Chat Configrations
  3. Type "chat" in widget content box without quote in small letters (Case Senstive)

Database Setup (For Buyers Only)

This is the main setup where all chat conversation is stored for every Punjab Press user

Use Public Database

If you want to use public Database instead of creating or using your own then read below:

  1. On Blogger Dashboard Click Layout
  2. Click on DATABASE Widget below Chat Configrations
  3. Type "punjabpressbt" in widget content box without quote in small letters (Case Senstive)
  4. Then Click Save

Your Own Database

If you want to create your own database then read below steps carefully:

  1. Go to Firebase.com and create an free account.
  2. Then Setup your first App.
  3. Enter app name and App URL Name but make sure your App URL Name is your Database
  4. Then click on Create New App.
  5. Done.
  6. On Blogger Dashboard Click Layout
  7. Click on DATABASE Widget below Chat Configrations
  8. Type your App URL Name as Database in widget content box without quote in small letters (Case Senstive)
  9. Then Click Save

Auto Featured Slider



Simple Setup

To add Featured Post widget follow this step

  1. On Blogger Dashboard Click Layout
  2. Click Add a Gadget
  3. Choose HTML/Javascript widget
  4. Copy this Code:
    5. <div id="featpost"></div> <script type='text/javascript'> jQuery("#featpost").AutofeaturedPost({ MaxPost:5 }); </script>
  5. Then Click Save

Advanced Setup

(Default) Complete Javascript to call Auto Featured Post like this code: <div id="ID_Place_To_Load_Plugin"></div> <script type='text/javascript'> jQuery("#ID_Place_To_Load_Plugin").AutofeaturedPost({ blogURL: "", MaxPost: 5, ImageSize: 600, ShowDate: true, ShowComment: true, pagination: true, thumbnails: true, loader:"pie", height:"50%", Random: true, tagName: false }); </script>

Default javascript mean if you just insert code like this: <div id="ID_Place_To_Load_Plugin"></div> and jQuery("#ID_Place_To_Load_Plugin").AutofeaturedPost();

Featured post result will same with default script.

if you want to change it. ex: Show featured with maximal 8 from label Movies, don't show date and comment, and post not Random (Recent). Yours script will look like this: jQuery("#ID_Place_To_Load_Plugin").AutofeaturedPost({ MaxPost: 8, ShowDate: false, ShowComment: false, RandompostActive: false, tagName: "Movies" });

How to use Codes?

Property Description
blogURL Yours blog URL or you can leave it blank to load where this script load.
MaxPost Maximum number artikel will show on Featured Post
ImageSize Post Image size (in pixel)
ShowDate If you want to show post date (true or false)
ShowComment If you want to show Comment number (true or false)
pagination If you want to show slideshow pagination
thumbnails If you want to show slideshow Thumbnails
loader kind of loader style : "pie", "bar", "none"
height Slideshow height : "50%" mean if will resize 50% from slideshow with if you want make it fix to 300px just change with "300px"
Random default true. It's will show article ramdomly. If want to show Recent Post just change to false
tagName If you want to show Featured Post by the Label. Example: to show featured post from Sports Label just write like this tagName:"Sports"

animated make it have animation effect (for Vertical and Horizontal Post)
sliderType make it show carousel slide show or Slide show : "c" (Carousel Slider) or "f" (full Slideshow Slider)
animation Full Slideshow effect : "slide" or "fade"
tagName If you want to show Post by the Label. Example: to show post from Sports Label just write like this tagName:"Sports"

How to Get Facebook App ID?

  1. Go to http://developers.facebook.com and login with your Facebook account.
  2. If you don't already have a Facebook Developer account, click on the "Apps" button in the top title bar of the web page. Follow the steps in the wizard to register as a Facebook developer.
  3. Once you're registered as a Facebook developer, click on the "Apps" button in the top title bar of the web page. Click on the "Create New App" button in the drop down menu to create a new Facebook app.
  4. Choose the "Website" option on the right side of this window.
  5. Choose a name for your app and click on the "Create New Facebook App ID" button.
  6. Choose a category for your app and click the "Create App ID" button.
  7. Scroll to the bottom of the next page and fill in a valid Site URL and Mobile Site URL. If you don't have a website or mobile site URL, just use http://www.example.com. Once you have entered your information here, click "Next".
  8. Scroll down the page and click the hyperlink "Skip to Developer Dashboard"
  9. Once the Facebook app has successfully been created, you still need to set the app live before the necessary key (App ID and App Secret)
  10. Go to the "Settings" page of your Facebook app and make sure that the Contact Email field is filled in. The app can't be set live without a Contact Email
  11. Once you've entered your Contact Email, you can set the app live in the "Status & Review" page of the app.
  12. Go back to the "Dashboard" of the app. Here you will find the App ID
  13. Done
