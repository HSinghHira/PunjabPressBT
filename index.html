Installation

This Theme is made for Blogger platform only. Theme installation on blogger is very easy task. A beginner with Zero knowledge can easily install theme on Blogger.

Step By Step

Here is a brief step by step on how to install Blogger Theme:

  1. Unzip the “Punjab Press.zip”.
  2. It will contain “Punjab Press.xml” File.
  3. Drop it to Desktop.
  4. On Blogger Dashboard Click Template.
  5. Then click Backup/Restore (Top Right).
  6. Now Choose File
  7. Upload “Punjab Press.xml” file from Desktop.
  8. Then Click Upload.
  9. Done.

Auto Featured Slider



Simple Setup

To add Featured Post widget follow this step

  1. On Blogger Dashboard Click Layout
  2. Click Add a Gadget
  3. Choose HTML/Javascript widget
  4. Copy this Code:
    5. <div id="featpost"></div> <script type='text/javascript'> jQuery("#featpost").AutofeaturedPost({ MaxPost:5 }); </script>
  5. Then Click Save

Advanced Setup

(Default) Complete Javascript to call Auto Featured Post like this code: <div id="ID_Place_To_Load_Plugin"></div> <script type='text/javascript'> jQuery("#ID_Place_To_Load_Plugin").AutofeaturedPost({ blogURL: "", MaxPost: 4, ImageSize: 500, Summarylength: 150, slideSpeed:5000, ShowDate: true, ShowComment: true, animation:"slide", NoCommentText: "No Comment", OneCommentText: "Comment", CommentText: "Comments", RandompostActive: true, MonthNames: ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct", "Nov", "Dec"], pBlank: "http://1.bp.blogspot.com/-htG7vy9vIAA/Tp0KrMUdoWI/AAAAAAAABAU/e7XkFtErqsU/s72-c/grey.gif", tagName: false }); }); </script>

Default javascript mean if you just insert code like this: <div id="ID_Place_To_Load_Plugin"></div> and jQuery("#ID_Place_To_Load_Plugin").AutofeaturedPost();

Featured post result will same with default script.

if you want to change it. ex: Show featured with maximal 8 from label Movies, don't show date and comment, and post not Random (Recent). Yours script will look like this: jQuery("#ID_Place_To_Load_Plugin").AutofeaturedPost({ MaxPost: 8, ShowDate: false, ShowComment: false, RandompostActive: false, tagName: "Movies" });

How to use Codes?

Property Description
blogURL Yours blog URL or you can leave it blank to load where this script load.
MaxPost Maximum number artikel will show on Featured Post
Summarylength Number of summary characters to show.
slideSpeed Set the speed of the slideshow cycling, in milliseconds
ImageSize Post Image size (in pixel)
ShowDate If you want to show post date (true or false)
ShowComment If you want to show Comment number (true or false)
animation Select your animation type, "fade" or "slide"
NoCommentText Text if your post don't have comment
OneCommentText Text if your post have one comment
CommentText Text if your post have more than one comment
RandompostActive default true. It's will show article ramdomly. If want to show Recent Post just change to false
pBlank Backup image if post don't have image store on blogspot or picasa web
MonthNames Format months name
tagName If you want to show Featured Post by the Label. Example: to show featured post from Sports Label just write like this tagName:"Sports"

Manual Slide Show (Using Image Widget)



Simple Setup

To Create Manual Slide Show follow this step

  1. On Blogger Dashboard Click Layout
  2. On its Area Click Add a Gadeget
  3. Choose Image Widget
    • Fill With Image Title
    • Image Description
    • Link When Image or Title Click
    • Your Image Link or upload
    • Uncheck shrink to fit
  4. Then Click Save

1. Recommended Image Size 670px x 280px.
2. If you want to Show 5 Images in Slideshow then add Image Widget 5 times with different Images and Links.

Advanced Setup

  1. If you want to Change Slide Show Speed or Animation Type then go to Template
  2. Click Edit HTML
  3. Scroll down and find this code:
    4. manualfeaturedPost();
  4. Replcae above Script with this Script:
    5. manualfeaturedPost({ slideSpeed:5000, animation:"slide" });
  5. Then Click Save Template

How to use Codes?

Property Description
slideSpeed Set the speed of the slideshow cycling, in millisecond
MaxPost Select your animation type, "fade" or "slide"
